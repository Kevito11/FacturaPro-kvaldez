# FacturaPro - Sistema de Gestión Empresarial

**FacturaPro** es una plataforma de facturación e inventario de nivel "Enterprise", diseñada para ofrecer una experiencia operativa de alto rendimiento con una interfaz moderna y fluida.

## 🚀 Características Principales

*   **Gestión Documental Completa**: Emisión ágil de Facturas, Pedidos, Notas de Crédito y Notas de Débito.
*   **Búsqueda Inteligente**: Selectores premium que permiten filtrar Clientes y Productos instantáneamente por ID o Nombre.
*   **Control de Inventario Avanzado**: Fichas de productos con Código de Barras y **cálculo automático de rentabilidad** (Margen de utilidad en RD$ y %).
*   **Directorio CRM**: Gestión detallada de clientes con límites de crédito, contactos y perfiles fiscales (RNC/Cédula).
*   **Diseño Premium**: Interfaz inmersiva basada en *Glassmorphism*, gradientes profundos y diseño responsivo.

## 🛠️ Stack Tecnológico

*   **Frontend**: React.js, TypeScript, Vite, Tailwind CSS.
*   **Backend**: Node.js, Express.
*   **Base de Datos**: Microsoft SQL Server (MSSQL).
*   **Iconografía**: Lucide React.
